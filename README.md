# max-app-proof

Minimal React + Vite proof application prepared for GitHub Pages deployment.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Deployment

This app is configured for GitHub Pages at:

- repo: `MAX-elitendt/max-app-proof`
- hosted URL: `https://max-elitendt.github.io/max-app-proof/`

Deployment is handled by `.github/workflows/deploy-pages.yml` on pushes to `main`.
